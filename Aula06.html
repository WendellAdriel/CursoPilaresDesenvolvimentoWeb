Aula 06 - Tabelas

Tags para utilizar tabelas

  • A tag "table" é usada para definir uma tabela
  • A tag "thead" é usada para definir o cabeçalho da tabela
  • A tag "tbody" é usada para definir o corpo da tabela
  • A tag "tfoot" é usada para definir o rodapé da tabela
  • A tag "th" é usada para definir uma célula de título
  • A tag "tr" é usada para definir uma linha em uma tabela
  • A tag "td" é usada para definir uma célula em uma tabela
