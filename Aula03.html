Aula 03 - Tags

Tags

  • Base do HTML
  • Containers que informam aos Browsers quais elementos estão sendo utilizados na página
  • Toda TAG inicia com < e termina com >

Atributos das Tags

  • São as propriedades de uma tag
  • Cada tag pode ter vários atributos
  • Os atributos não são iguais para todas as tags
  • Ajudam a "configurar" a tag

Tags Header

  • Seis tags header: h1 ao h6
  • H1 é o Header mais importante
