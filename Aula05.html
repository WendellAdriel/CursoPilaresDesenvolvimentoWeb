Aula 05 - Links

Tag de Hiperlink

  • A tag "a" é utilizada para criação de hiperlinks
  • Hiperlinks são usados para conectar uma página a outra
  • Os links tem uma formatação padrão que pode ser modificada através do CSS

Atributos mais usados da Tag de Hiperlink

  • href: utilizado para especificar a URL da página que o link irá acessar
  • download: utilizado para especificar que o objeto alvo será baixado quando o usuário clicar no link
  • target: utilizado para especificar onde o documento linkado será aberto
