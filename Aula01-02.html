Olá Mundo!

Conceitos importantes

Navegadores

  • Navegador = Browser = Cliente
  • Fazem requisições para algum(ns) servidor(es)
  • Recebem instruções HTML e convertem em uma página renderizada
  • Utiliza recursos da rede

Servidores

  • Espera pelas requisiões dos "clientes"
  • Trabalha a requisição feita e responde
  • Pode se comunicar com outro servidor se necessário
  • Fornece recursos de rede

Protocolos

  • HTTP (Hypertext Transfer Protocol) - base para comunicação na World Wide Web. Protocolo de requisição-resposta
  • HTTPS (Hypertext Transfer Protocol Secure) - implementação do protocolo HTTP com uma camada de segurança adicional. Esta camada faz com que os dados sejam transmitidos de forma criptografada e que também verificar autenticidade de servidores por meio de certificados digitais.
  • Existem outros protocolos que iremos abordar durante o curso
