Aula 07 - Formulários - Parte 01

Formulários

  • Meio de interação do usuário com nossa aplicação
  • Usado para coletar informações do usuário que sejam importantes para nossa aplicação
  • Um dos recursos mais usados em aplicações e sites

Atributos mais usados da Tag de Formulário

  • action: onde as informações do formulário serão enviadas
  • autocomplete: define se os campos do formulário poderão ter autocomplete
  • enctype: define como os dados do formulário devem ser codificados, usado principalmente quando o usuário pode fazer upload de arquivos. Só pode ser usado com o método POST
  • method: define o método HTTP usado para enviar os dados do formulário
  • name: define um nome para o formulário
  • novalidate: define que o formulário não deve ser validado quando for enviado
